# create-mincu

[Mincu - 南大家园](https://github.com/ncuhome/mincu)

## Quick Start

快速创建 mincu 项目：

```sh
# npm
$ npm init mincu@latest
# yarn
$ yarn create mincu
# pnpm
$ pnpm create mincu
# or
$ npx create-mincu
```
